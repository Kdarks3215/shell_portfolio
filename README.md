# Terminal Portfolio

Single-page DevOps/System Admin portfolio with a Bash-like terminal UI.

## Local Run

Option A: open `index.html` directly in a browser.

Option B: serve locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.



## Customize

- Edit profile data in `app.js` (look for the `profile` object at the top).
- Add or modify commands in `app.js` within `registerCommands()`.
